# Marketing & E-Commerce Analytics (SQL + Power BI)

## Overview
End-to-end analytics project analyzing user behavior, marketing performance, funnel conversion, A/B testing, and customer & product insights for an e-commerce business.

---

## Business Problems Solved
- Where do users drop off in the purchase funnel?
- Which campaigns and channels drive real revenue?
- Do experiments improve conversion and revenue?
- Which customers and products generate long-term value?

---

## Tools & Technologies
- SQL (MySQL)
- Power BI
- Data Modeling & Analytics Views
